Ubuntu常用命令

常用命令

命令 含义
sudo apt install 软件名 安装软件命令（有root权限）
sudo apt remove 软件名 删除软件命令（有root权限）
sudo nautilus 打开文件（有root权限）
su root 切换到“root”
ls 列出当前目录文件（不包括隐含文件）
ls -a 列出当前目录文件（包括隐含文件）
ls -l 列出当前目录下文件的详细信息
cd … 回当前目录的上一级目录
cd - 回上一次所在的目录
cd ~ 或 cd 回当前用户的宿主目录
kill 进程ID或进程名 终止该进程
